## 1. 브랜드 아이덴티티

### 사이트명
**쫀득연구소**

### 슬로건
> "오늘의 실험: 두쫀쿠 만들기"

### 브랜드 스토리
연구소라고 해서 어렵고 딱딱한 곳이 아니다. 
주방에서 마시멜로우 녹이면서 "이게 되나?" 싶은 마음으로 시작하는 곳.
실패해도 괜찮고, 대충 해도 맛있으면 성공인 그런 연구소.

### 톤앤매너

| 이것 | 이것 아님 |
|------|-----------|
| "카다이프, 처음 들어봤죠?" | "카다이프는 중동 지역의 전통 식재료로..." |
| "망해도 맛은 있어요" | "누구나 쉽게 성공할 수 있습니다!" |
| "연구 결과: 냉동 30분 필수" | "TIP! 냉동실에 30분 넣어주세요~" |
| "피스타치오, 왜 이렇게 비싼 거야" | "고급 재료로 특별한 디저트를 만들어보세요" |

### 말투 예시
- "오늘의 연구 주제"
- "실험 준비물"
- "연구 결과 보고"
- "이건 좀 의외였음"
- "결론: 해볼 만함"

## 3. 비주얼 디렉션

### 일러스트 스타일

**캐릭터: 쫀득이 (가칭)**
- 마시멜로우를 의인화한 캐릭터
- 흰색 몸체, 살짝 늘어난 형태 (쫀득한 느낌)
- 작은 눈, 단순한 표정
- 상황별 변형: 녹는 쫀득이, 놀란 쫀득이, 뿌듯한 쫀득이

**일러스트 톤**
- 선: 손그림 느낌의 불규칙한 선
- 채색: 플랫, 단색 위주
- 디테일: 최소화 (복잡하지 않게)
- 참고: 카카오프렌즈보다 단순, 라인프렌즈보다 따뜻함

**재료 일러스트**
- 실제 패키지가 아닌 단순화된 형태
- 각 재료의 특징만 살림
  - 카다이프: 노란 면발 뭉치
  - 피스타치오: 초록 알갱이
  - 마시멜로우: 통통한 흰 덩어리

### 컬러 팔레트

| 이름 | 컬러 | 용도 |
|------|------|------|
| 쫀득 화이트 | #FFF8F0 | 배경, 여백 |
| 피스타치오 | #A8D5A2 | 메인 포인트 |
| 코코아 | #5D4037 | 텍스트, 강조 |
| 버터 크림 | #F5E6C8 | 카드 배경 |
| 캐러멜 | #D4A574 | 서브 포인트 |

### 타이포그래피
- 제목: 둥근 고딕 (예: Pretendard, 고딕 A1)
- 본문: 가독성 좋은 본문용 (16px 이상)
- 포인트: 손글씨 느낌 (인포그래픽 내 코멘트용)

## 5. 콘텐츠 상세

### 5-1. 오늘의 연구 (메인 레시피)

**형식**: 세로 스크롤 인포그래픽 (한 장짜리 이미지 아님)

**구성**:
```
[헤더]
오늘의 연구: 두쫀쿠 만들기
난이도: ★★☆☆☆
소요시간: 약 1시간
결과물: 10개

[실험 준비물]
재료 일러스트 + 분량
(체크박스 형태로 준비 확인)

[실험 과정]
Step 1 → Step 2 → Step 3 → Step 4
(각 단계별 일러스트 + 한 줄 설명)

[연구 메모]
"여기서 망하는 이유" 박스
쫀득이 캐릭터가 코멘트하는 형식

[실험 결과]
완성된 두쫀쿠 일러스트
"수고했어요" 메시지
```

**UX 포인트**:
- 한 화면에 한 단계씩
- 스크롤할 때마다 다음 단계로 자연스럽게 이동
- 각 단계마다 쫀득이의 한마디

---

### 5-2. 재료 탐구생활

**목적**: "이게 뭔데?" 싶은 재료들을 친근하게 소개

**형식**: 카드형 콘텐츠 (SNS 카드뉴스 느낌)

**예시: 카다이프 편**

```
[카드 1]
"카다이프요?"
"네, 카다이프요"
(물음표 띄운 쫀득이 일러스트)

[카드 2]
카다이프는요
- 밀가루로 만든 아주 가는 면
- 중동에서 디저트에 많이 씀
- 한국에선 생소하지만 터키에선 흔함
(면발 일러스트)

[카드 3]
생김새
- 실타래처럼 엉켜있음
- 색깔은 연한 노란색
- 만지면 푸석푸석
(실제 형태 단순화한 일러스트)

[카드 4]
맛은요
- 그냥 먹으면 밀가루 맛
- 구우면 바삭 + 고소
- 버터랑 굽는 게 핵심
(굽기 전/후 비교 일러스트)

[카드 5]
어디서 사요?
- 쿠팡: "카다이프" 검색
- 제과재료몰: 브레드가든, 아이엠베이커
- 가격: 500g에 1만원 내외
(쇼핑 아이콘)

[카드 6]
없으면요?
- 콘플레이크 (바삭함 대체)
- 쌀국수면 튀긴 것 (비슷한 식감)
- 누가크래커 부순 것
(대체재 일러스트)

[카드 7]
결론
"처음 보는 재료지만
생각보다 구하기 쉬움"
(엄지 든 쫀득이)
```

**다른 재료 편 주제**:
- 피스타치오 스프레드: "왜 이렇게 비싼 거야" (가격 이유, 브랜드 비교)
- 마시멜로우: "다 같은 마시멜로우 아니었음" (종류별 차이)
- 탈지분유: "편의점에서 살 수 있음" (우유스틱 대체)
- 화이트 커버춰: "코팅초코랑 다른 거임" (차이점 설명)

### 5-3. 연구 결과 보고서 (인포그래픽)

**형식**: 한 장 또는 두 장짜리 인포그래픽 이미지

**예시 1: 두쫀쿠 해부도**
```
[두쫀쿠 단면 일러스트]

겉 → 코코아파우더 (달라붙음 방지)
     ↓
중간 → 마시멜로우 반죽 (쫀득함 담당)
     ↓
속 → 피스타치오 + 카다이프 (고소함 + 바삭함)

"세 가지 식감이 한입에"
```

**예시 2: 비용 분석**
```
[직접 만들기 vs 사먹기]

10개 기준

직접 만들기: 약 22,000원
├─ 카다이프: 3,000원
├─ 피스타치오 스프레드: 12,000원
├─ 마시멜로우: 2,000원
├─ 기타: 5,000원
→ 개당 2,200원

사먹기: 약 70,000~100,000원
→ 개당 7,000~10,000원

결론: 직접 만들면 3배 이상 저렴
(단, 시간과 노력은 별도)
```

**예시 3: 실패 원인 TOP 5**
```
[두쫀쿠 실패 유형]

1위. 마시멜로우가 딱딱해짐 (42%)
→ 원인: 너무 식혀서
→ 해결: 따뜻할 때 빨리 작업

2위. 속이 터짐 (28%)
→ 원인: 속을 덜 얼림
→ 해결: 30분 이상 냉동

3위. 코코아가 뭉침 (15%)
→ 원인: 손에 물기
→ 해결: 완전히 건조 후 작업

4위. 카다이프가 눅눅 (10%)
→ 원인: 덜 구움 / 습기
→ 해결: 황금빛 될 때까지 + 밀폐보관

5. 맛이 안 남 (5%)
→ 원인: 재료 문제
→ 해결: 커버춰 사용, 스프레드 브랜드 변경
```

### 5-4. 연구 일지 (가벼운 읽을거리)

**목적**: 정보 전달보다 공감, 재미

**형식**: 짧은 글 + 일러스트 (블로그 포스팅 아님)

**예시 주제**:

| 제목 | 내용 요약 |
|------|-----------|
| "피스타치오 100개 깠음" | 껍질 있는 피스타치오 사서 후회한 이야기 |
| "마시멜로우 녹이다 팔 아픔" | 계속 저어야 하는 고통 |
| "다이소 마시멜로우 품절임" | 재료 구하기 대란 현황 |
| "냉동실에서 잊어버림" | 3일 후 발견한 피스타치오 볼 |
| "가족한테 먹였더니" | 반응 모음 (솔직 후기) |

**톤 예시**:
```
[연구 일지 #3]
"마시멜로우, 생각보다 빨리 굳음"

마시멜로우 녹이고 나서
"사진 한 장 찍어야지" 했다가
30초 만에 굳어버림.

다음부터는 사진 포기하기로 함.
교훈: 녹으면 바로 작업할 것.

(굳어버린 마시멜로우 앞에서 멍한 쫀득이 일러스트)
```

## 6. 인포그래픽 가이드라인

### 원칙
1. **한 장에 하나의 메시지**: 정보 과부하 금지
2. **텍스트 최소화**: 읽는 게 아니라 보는 것
3. **비교/대조 활용**: 숫자보다 시각적 차이
4. **여백 충분히**: 빽빽하면 안 봄

### 구성 요소
- 제목: 크고 명확하게
- 핵심 숫자: 가장 눈에 띄게
- 부연 설명: 작게, 필요한 만큼만
- 일러스트: 이해를 돕는 용도
- 출처/주석: 하단에 작게

### 피해야 할 것
- 3D 그래프
- 그라데이션 과다 사용
- 장식용 아이콘 남발
- 텍스트로 가득 찬 인포그래픽
